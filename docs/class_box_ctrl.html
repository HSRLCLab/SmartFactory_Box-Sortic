BoxCtrl::Event BoxCtrl::doAction_calculateOptVehicle ( )
private

main action of the calculateOptVehicle

check all saved messages.
select all vehicle in same sector and choose the nearest one.
if the nearest one is valid update box.req with id of the nearest vehicle
and return Event::CalcOptVal.
if not valid return Event::NoAnswerReceived.
