void checkIfAckReceivedfromResponses ( )

FSM runs until acknoledgement of desired Vehicle arrived, check if right Vehicle answered to get SmartBox transported.

If the right vehicle answer change state to checkIfTransporedfromResponses()
Else if desired vehicle doesn't respond ask the nexst one, change state back to Else no vehicle responded, chaneg state to publishLevel()

◆ loop()

void loop ( )

one loop per cycle (SB full -> transported -> returned empty)

After creating a setup() function, which initializes and sets the initial values, the loop() function does precisely what its name suggests, and loops consecutively, allowing your program to change and respond. Use it to actively control the board.

void loopEmpty ( )

FSM loop until Box full.

if empty leave LED off or turn it off

only subscribe once but publish repeatedly

if the Box is full turn the LED on
